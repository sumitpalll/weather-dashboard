# Weather Dashboard

## Live Demo
https://sumitpalll.github.io/weather-dashboard/

## Features

### Home Page
- Current Location (Latitude & Longitude)
- Temperature Graph
- Humidity Graph
- Precipitation Graph
- Wind Speed Graph
- Air Quality (PM2.5 / PM10)

### History Page
- Past Days/Years Weather Data
- Min / Max Temperature
- Precipitation Summary
- Wind Speed
- Air Quality (PM2.5 / PM10)

## Graph Features
- Horizontal Scrolling
- Zoom Functionality
- Responsive Design (Mobile Friendly)

## Tech Stack
- React.js
- Vite
- Recharts
- Open-Meteo API

## Setup Instructions

```bash
git clone https://github.com/sumitpalll/weather-dashboard.git
cd weather-dashboard
npm install
npm run dev
```
